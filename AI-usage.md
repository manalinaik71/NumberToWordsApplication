
# AI Usage Transparency
I used ChatGPT as a support tool while working on this application.
I mainly used it to discuss ideas, and clarify some technical points in terms of number to words conversion. The actual coding, testing, debugging, and final decisions were done by me.

## How I used AI assistance
### I used ChatGPT for the following areas:
*	To understand how to split larger numbers into millions, thousands, and hundreds.
*	To clarify how cents should be handled, especially cases like 25,1, which should be treated as 25 dollars and 10 cents.
*	To understand the difference between English and German number wording, especially for million numbers and thousands.
*	To get help with the README structure.

## Example prompts discussed
### Below topics I discussed with ChatGPT:
*	How to convert numbers into words if number is in millions or thousands in C#.
*	How to handle bigger numbers like 90500 and 888888888.
*	How to handle number conversion in German like 90500 and 888888888.
*	How to write a clear README file.

## Final note
AI was used only as a learning and support tool. I implemented the application myself, tested the functionality, fixed the issues, and finalized the solution.
